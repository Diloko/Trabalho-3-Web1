HTML Classes

Todos os documentos HTML começam com a declaração do tipo <!DOCTYPE html>

O documento HTML começa com <html> e termina com </html>

A parte visível do documento HTML fica entre <body> e </body>

Exemplo de código abaixo

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
