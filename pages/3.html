HTML Attributes

  • Todo elemento HTML pode ter atributos
  • Atributos nos dão informações adicionais sobre os elementos
  • Atributos sempre são especificados na tag inicial
  • Atributos usualmente aparecem no par name/value tipo: name="value"

    • O atributo href

    A tag <a> define um hyperlink. O atributo href específica uma URL de página de onde o link deve direcionar

    Exemplo:

    <a href="http://www.uol.com.br">Visite a página do Uol para ler as últimas notícias</a>

    A tag <img> é usada para incorporar uma imagem em páginas HTML. O atributo src específica o caminho da imagem a ser exibida

    <img src="Logo_fatec.png">
