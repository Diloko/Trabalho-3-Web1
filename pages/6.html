HTML Styles

O HTML style é um atributo para adicionar estilo ao elemento, como cor, fonte, tamanho e etc

A sintaxe base do style é <tagname style="property:value;"> onde property e value são dados do css que veremos no futuro

Alguns exemplos de uso do atributo style abaixo

Para alterar a cor do fundo de um heading ou parágrafo, podemos usar um código com a seguinte estrutura:
<h3 style="background-color:rgb(80, 8, 62);">Insira o título aqui</h3>

Título com fundo


Já para alterar a cor do texto, podemos usar um código com a seguinte estrutura:
<h3 style="color:red;">Insira o texto aqui</h3>
